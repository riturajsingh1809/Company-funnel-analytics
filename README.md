# Company Funnel Analytics

## Project Overview
This project was developed as part of a Data Analyst Internship Task.

The objective of this project is to analyze the company recruitment funnel using sample data and generate business insights.

---

## Tools Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Microsoft Excel
- GitHub

---

## Dataset

The dataset contains sample student recruitment records with the following columns:

- Student_ID
- Job_ID
- Search_Date
- Job_Category
- Match_Score
- Viewed
- Applied
- Shortlisted
- Hired

---

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. KPI Calculation
5. Company Funnel Analysis
6. Data Visualization
7. Report Generation

---

## KPIs

- Total Students
- Jobs Viewed
- Applications
- Shortlisted Candidates
- Hired Candidates

## Visualizations
